FLODE Visueller Flow

Du verbindest Knoten mit der Maus —
Trigger → Bedingung → Aktionen.

Kein YAML tippen. Kein Syntax-Raten.
Einfach zeichnen und speichern.

Output automation.yaml 
# generiert von FLODE
alias: Abend-Licht
trigger:
  - platform: sun
    event: sunset
condition:
  - condition: state
    entity_id: group.family
    state: not_home
action:
  - service: light.turn_off
    target:
      entity_id: all

Funktionen

Alles was du brauchst.
Nichts was du nicht brauchst.

FLODE ist ein Fork von C.A.F.E. mit Dutzenden Bugfixes und neuen Features.

🎯

Visueller Flow-Editor

Trigger, Bedingungen und Aktionen per Drag-and-Drop auf einer Leinwand anordnen und verbinden.

📄

100 % natives YAML

Kein proprietäres Format. Die Ausgabe ist Standard HA-Automatisierungs-YAML — kein Vendor Lock-in.

🔄

Bidirektional

Bestehende Home Assistant Automatisierungen importieren, visuell bearbeiten und direkt zurückspeichern.

🐛

Trace-Integration

Flows mit der offiziellen HA Trace-Ansicht debuggen. Sieh genau, welcher Pfad ausgeführt wurde.

🔀

State Machines

Komplexe Schleifen und Verzweigungen via automatischem State-Machine-Muster — ohne Umwege.

🌍

DE & EN, Dark & Light

Vollständige i18n-Unterstützung. Dark und Light Mode folgen automatisch dem eingestellten HA-Theme.

Knotentypen

Vier Bausteine.
Unbegrenzte Möglichkeiten.

Jeder Knoten repräsentiert einen HA-Konzept direkt im visuellen Flow.

Trigger
Was startet es?
Zustand, Zeit, Ereignis, Webhook — jede HA-Trigger-Plattform.
Bedingung
Wann läuft es?
Filter — die Automatisierung läuft nur weiter, wenn die Bedingung erfüllt ist.
Aktion
Was passiert?
Dienst aufrufen, Ereignis auslösen, Verzögerung, Script aufrufen, …
Logik
OR / AND / NOT
Mehrere Bedingungen und Trigger gruppieren und logisch verknüpfen.

Installation

In 5 Minuten startklar.

Via HACS (empfohlen)

  1. 1 HACS öffnen → Integrationen → ⋮ → Benutzerdefinierte Repositories
  2. 2 URL https://github.com/SH1FT-W/flode eingeben, Typ Integration wählen → Hinzufügen
  3. 3 HACS → Integrationen → Nach FLODE suchen → Installieren
  4. 4 Home Assistant neu starten
  5. 5 Einstellungen → Integrationen → Integration hinzufügen → FLODE auswählen
